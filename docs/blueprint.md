# **App Name**: VoltCare

## Core Features:

- User Authentication: Secure login/signup functionality with username/phone/email, password, and 'show/hide' option.
- Complaint Submission: Users can submit complaints by specifying utility type (electricity, water), location details (town, ward, address), and problem category (power cuts, outages, etc.).
- Complaint Tracking: View history of submitted complaints with status updates (Submitted → In Progress → Resolved).
- Bill Payment Dashboard: Dashboard to view utility bill payments, including bill amount, payment status, due date, and options to 'Pay Now' or 'View Bill Details'.
- Payment History: Display a list of past payments with options to view details and download receipts. Filter and sort by utility type, department, status, and date.
- Profile Management: User profile with picture, name, username, and options to edit personal details and language.
- AI Powered energy consumption tips: AI tool analyzes your energy usage data, comparing against similar households, and suggests ways to optimize for better consumption and reduce energy waste.

## Style Guidelines:

- Primary color: Neon blue (#13E2F8) for electric energy and a futuristic feel.
- Background color: Dark gray (#282929) for a sophisticated, energy-themed backdrop.
- Accent color: Soft gold (#F0D239) to highlight key interactive elements and provide contrast against the dark background.
- Body and headline font: 'Inter' sans-serif font, providing a modern and accessible style.
- Use electricity-themed icons to match functionalities, in glowing neon blue.
- Mobile-first layout with glassmorphism effect on cards, forms, and navigation elements, incorporating rounded corners and a clean, accessible design.
- Subtle animations for page transitions, button presses, and status updates to enhance user engagement.
